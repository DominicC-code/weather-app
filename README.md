# Weather App

A desktop weather application built with PyQt6 that displays real-time weather information for any city with temperature unit conversion.

## Features:
- Search weather by city name
- Real-time data from OpenWeatherMap API
- Temperature display in Fahrenheit or Celsius
- Weather emoji indicators (sun, rain, snow, etc.)
- Detailed weather descriptions
- Error handling for invalid cities and connection issues

## How to use:
1. Install required packages:
```
   pip install PyQt6 requests
```
2. Run `python weather-app.py`
3. Enter a city name
4. Click "Get Weather"
5. Toggle between °F and °C using the buttons

## What I learned:
- Building GUI applications with PyQt6
- Making API requests with the requests library
- Parsing JSON data from APIs
- Exception handling for network errors
- Temperature unit conversions
- Dynamic UI updates based on user input
